~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
~~~~~~~~~ ONE PASS ASSEMBLER ~~~~~~~~~~~
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~



Program usage (on command prompt):
assembler.exe [SOURCE FILE]

Example (on command prompt):
assembler.exe sample_input.txt

Output will be available in the same folder with the name "objfile.txt"
